Primary action button for Augur — large, high-contrast touch targets for one-handed table-side use.

```jsx
<Button variant="primary" size="lg" block>Lancer la simulation</Button>
<Button variant="secondary" icon={<i data-lucide="dices" />}>Relancer</Button>
<Button variant="ghost">Annuler</Button>
<Button variant="danger">Marquer en fuite</Button>
```

Variants: `primary` (signal cyan), `secondary` (surface), `ghost` (text-only), `danger` (flee-red outline). Sizes `md` (52px) / `lg` (60px). Press = scale 0.97 + slight darken.
