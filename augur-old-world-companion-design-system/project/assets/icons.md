# Augur — iconography working set (Lucide)

Loaded from CDN: `<script src="https://unpkg.com/lucide@latest"></script>` then `lucide.createIcons()`.
Rendered as `<i data-lucide="name"></i>`. Stroke ~2px, `currentColor`, line style (never filled).

| Purpose | Lucide name |
|---|---|
| Combat / engagement | `swords` |
| Defence / armour save | `shield`, `shield-half` |
| Roll / probability | `dices` |
| Banner / standard | `flag` |
| General (leader) | `crown` |
| Increment / decrement | `plus`, `minus` |
| Step forward / nav | `chevron-right`, `chevron-left`, `chevron-up` |
| Victory points | `target` |
| Unit strength | `users` |
| Fleeing / catastrophe | `wind`, `triangle-alert` |
| Destroyed | `skull` |
| Settings / scenario | `sliders-horizontal`, `map` |
| Toggle / done | `check`, `x` |

No emoji, no Unicode pictographs. Status communicated by coloured dot + word, never by glyph alone.
